# Checkmarx Tooling Library

Common shared tools and classes for Checkmarx PS tools

### Notice

Tooling developed using Python. Please use Python3!
## Python dependencies

See ".build/requirements.txt" for pip dependencies. 
Use "pip install -r .build/requirements.txt" to install them.

## Build tools

See ".build/build_requirements.txt"
Install the dependencies for the build process.

## Builders

The batch/bash files below are redy to use for the corresponding building environment. Just invike them in you environment.

- On MacOS          build-macos-sh
- On Linux          build-ubuntu.sh
- On Windows        build-windows.bat
